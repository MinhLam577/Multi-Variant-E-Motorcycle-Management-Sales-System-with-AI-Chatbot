# HongSon Admin - Web Site

## 🚀 Giới Thiệu

HongSon Admin

## 📌 Cấu Hình Dự Án

Dự án sử dụng các công nghệ sau:

- **ReactJS**
- **Ant Design** (UI Components)
- **React Router** (Quản lý điều hướng)
- **Axios** (Gọi API)
- **Modx** (Quản lý trạng thái)
- **Yarn** (Trình quản lý gói)

## 🛠 Cài Đặt & Chạy Dự Án

### 1️⃣ **Cài đặt Yarn**

Nếu bạn chưa cài đặt Yarn, hãy chạy lệnh sau:

```sh
npm install -g yarn
```

### 2️⃣ **Clone dự án**

````sh
git clone git@github.com:OpenverseHQ/hongson-fe.git

### 3️⃣ **Cài đặt dependencies**

```sh
yarn install
````

### 4️⃣ **Chạy dự án**

```sh
yarn start
```

Dự án sẽ chạy tại `http://localhost:3000`

## 📁 Cấu Trúc Thư Mục

```
├── public/          # Chứa các file tĩnh
├── src/             # Source code chính
│   ├── components/  # Các component tái sử dụng
│   ├── pages/       # Các trang chính
│   ├── store/       # Mobx store
│   ├── api/         # Gọi API
│   ├── assets/      # Ảnh, icon, style
│   ├── App.js      # Component gốc
│   ├── index.js    # Entry point của ứng dụng
└── package.json     # Cấu hình dự án
```

## 🛠 Build Dự Án

Để build dự án:

```sh
yarn build
```
